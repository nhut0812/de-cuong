# Thư mục chứa file đề cương

Đặt các file PDF/DOCX đề cương vào đây.

## Hướng dẫn:

1. Tải file đề cương từ máy tính lên thư mục này
2. Hoặc dùng trang Admin để upload file
3. Tên file nên đặt theo format: `De-cuong-[Mon-hoc]-[Lop]-HK1.pdf`

## Ví dụ:
- De-cuong-Toan-5-HK1.pdf
- De-cuong-Tieng-Viet-5-HK1.pdf
- De-cuong-Khoa-hoc-5-HK1.pdf

## Lưu ý:
- Các file này chỉ là tên mẫu trong code
- Bạn cần upload file thật lên GitHub hoặc hosting khác
- Hoặc thay đổi `filePath` trong admin panel để trỏ đến link file thật
